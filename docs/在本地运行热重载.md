### 一、完整本地开发实施方案

#### 1. 修改 `app/lifecycle.py`（重要）

```python
from contextlib import asynccontextmanager
from fastapi import FastAPI

from app.services.ocr_service import ocr_service
from app.utils.logger import logger


@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动时初始化 PaddleX（只加载一次）
    logger.info("正在初始化 PaddleX OCR 模型...")
    ocr_service.initialize()
    logger.info("PaddleX OCR 模型初始化完成！")
    
    yield
    
    # 关闭时清理
    logger.info("服务关闭，清理资源...")
```

#### 2. 修改 `app/main.py`（保持干净）

```python
from fastapi import FastAPI

from app.api.health import router as health_router
from app.api.ocr import router as ocr_router
from app.lifecycle import lifespan
from app.config import APP_NAME

app = FastAPI(
    title=APP_NAME,
    version="1.0.0",
    lifespan=lifespan
)

app.include_router(health_router)
app.include_router(ocr_router)

@app.get("/")
async def root():
    return {"message": "OCR 服务已启动（本地热重载模式）"}
```

#### 3. 启动命令（关键）

在项目根目录执行：

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

推荐使用这个命令（更清晰）：

```bash
# Windows推荐使用这个
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

启动成功后你会看到：

```
Application startup complete.
Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
```

修改任何代码（包括 `business.py`、`layout.py`）后，服务会**自动重载**。

### 二、本地业务调用示例

在你的本地业务代码里这样调用：

```python
import requests

def ocr_local(image_path: str):
    with open(image_path, "rb") as f:
        response = requests.post(
            "http://127.0.0.1:8000/ocr",
            files={"file": f}
        )
    return response.json()

# 使用示例
result = ocr_local("test_business.jpg")
print(result)
```
